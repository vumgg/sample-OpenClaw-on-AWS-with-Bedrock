### Guardrail Word Policy 中文匹配的限制（重要）

Bedrock word policy 对中文使用**全文精确匹配**（非子串匹配），对英文使用**词边界匹配**。
- "新代币上线" 单独使用 → BLOCKED ✅
- "有什么新的token要上线吗"（中英混合）→ BLOCKED ✅（topic policy 语义触发）
- "下个月有没有新代币上线计划"（纯中文句子）→ NONE ❌（word policy 未匹配，topic policy 也未识别）
- "what tokens are we launching next quarter" → BLOCKED ✅

**结论**：纯中文的敏感问题依赖 topic policy 的语义识别，但当前 topic 定义是英文，对纯中文句子识别率偏低。如需更强的中文防护，需要在 word policy 里加更多变体，或将 topic definition 改成中英双语。

## Bug: Standard Runtime 500 — openclaw empty output on portal sessions

### 症状
- Playground 发消息返回 502/500：`RuntimeClientError: Received error (500) from runtime`
- CloudWatch 日志：`openclaw returned empty output (exit=0)`
- 所有 `port__` / `pt__` 前缀的 session 全部失败，`feis__` (飞书) 全部成功

### 根本原因
**`openclaw@latest` 在 2026-03-30 重建镜像时版本发生了变化。**

新版 openclaw 在 Gateway 模式下，response 通过 WebSocket 回 Gateway 交付，**不再写到 subprocess stdout**。`server.py` 的 `_invoke_openclaw_once` 用 `subprocess.run(capture_output=True)` 读 stdout，拿到空字符串，抛出 `RuntimeError("openclaw returned empty output")`。

飞书等 IM 渠道之所以成功，是因为：
- Feishu Gateway 插件因为没有配置凭证，立即以 1006 close 失败
- openclaw 捕获到 Gateway 失败，**主动回退到 embedded 模式**，将 response 写到 **stderr**
- `server.py` 在 stderr 里找到 `{` 开始的 JSON，解析成功

Portal (`pt__/port__`) 会话的 Gateway 连接成功但响应不走 stdout → 无法被捕获。

### 调查过程
1. 检查 CloudWatch：发现 `feis__` 成功，`port__` 全失败
2. 发现 `port__` 前缀触发 Gateway portal channel 回调 → 疑似死锁（26s hang）
3. 改 `tenant_router.py` 让 `portal → pt` 前缀 → 仍然 500，但变成 7s 快速失败
4. 发现 `pt__` session stderr 只有 `"sessionId": "..."` 两行，无 `{` → server.py 无法 fallback
5. 最终确认：问题是 **新版 openclaw** 行为变化，不是 prefix 问题

### 临时修复（回滚镜像）
```bash
# ECR latest 回滚到 March 29 23:30 的镜像 (fc45e6211a...)
aws ecr put-image --repository-name openclaw-multitenancy-multitenancy-agent \
  --image-tag latest --image-manifest <march29-manifest>

# 触发 Runtime 更新到 v35
aws bedrock-agentcore-control update-agent-runtime \
  --agent-runtime-id openclaw_multitenancy_runtime-olT3WX54rJ \
  --agent-runtime-artifact '{"containerConfiguration":{"containerUri":"...:latest"}}' \
  ...
```
Runtime 更新到 v35，使用 March 29 镜像，恢复正常。
